# Next.js + Cypress

example webapp using:

- [x] Typescript
- [x] Nextjs 
- [x] Cypress
- [x] Jest
- [x] JSDoc

Backend is generating mock data for this demo, but it's an AWS Lambda that will scale "for free", and could easily point to a genuine DB! :)
